I’ve been doing programming since the late 90’s. I started writing programs in Delphi. Later, I studied Java and worked in a team to create a web project in this language. The project involved using dynamic HTML and JavaScript on the client side.

In the early 2000s, I learned about the existence of PHP, which I really liked, and since then I have been developing web projects using this preprocessor on the server side, and JavaScript on the client side.

During that time, I created several projects: mostly for themselves, but there were works by contracts.

In recent years, I have begun to pay more attention to the frontend and more in-depth study of the features of JavaScript that appear in connection with the development of the language. I was also keen on Node.js as an alternative to PHP.

To date, I have about 20 years of experience in developing web applications for projects of various levels of complexity.
